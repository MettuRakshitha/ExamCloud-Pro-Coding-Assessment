🚀 ExamCloud Pro

> **A Smart Coding Assessment & Question Bank Management Platform for Faculty and Students**

ExamCloud Pro is a **full-stack web application** designed to simplify coding assessment management for educational institutions. It enables faculty to create, organize, assign, and monitor coding challenges and MCQ-based assessments through a centralized dashboard, while students can access assigned tasks and track their performance.

---

 🌟 Why ExamCloud Pro?

Traditional assessment systems often lack centralized coding challenge management and progress tracking. ExamCloud Pro solves this by providing a **single platform** where faculty can:

✔️ Assign coding challenges from **LeetCode, HackerRank, and external sources**  
✔️ Upload and manage **MCQ repositories in bulk**  
✔️ Monitor student progress and completion status  
✔️ Analyze assessment performance through interactive dashboards

 🏗️ Architecture

ExamCloud Pro follows a **Spring Boot MVC architecture** for scalable and maintainable development.

```text
Frontend (HTML/CSS/JS/Thymeleaf)
        ↓
Spring MVC Controllers
        ↓
Service Layer
        ↓
Hibernate / JPA
        ↓
MySQL Database
```
